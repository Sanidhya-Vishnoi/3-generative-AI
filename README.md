# 3-Generative-AI
A hands‑on collection of Python code demonstrating Azure AI capabilities across conversational AI, Retrieval‑Augmented Generation, autonomous agents & computer vision. Includes multi‑turn chatbots, RAG assistants, data‑analysis and support agents with tool calling, multi‑agent workflows & Vision/Face APIs for image analysis, OCR & facial attribute detection. Designed as practical, end‑to‑end examples of Azure AI Foundry, Azure OpenAI, Azure AI Search & Azure AI Vision services.
